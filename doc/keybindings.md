# Keybindings — example-window

Усі прив'язки клавіш визначено у `src/ecs/platform/inputmap.cpp` через `InputMap::createDefaultMap()`.  
Прив'язки можна змінити або розширити через `InputMap::addBinding()`.  
Режим **Numpad Emulation** та **Emulate 3-Button Mouse** вмикаються через `InputMap`.

---

## Рух камери

| Клавіша | Дія |
|---------|-----|
| `W` | Вперед |
| `S` | Назад |
| `A` | Вліво |
| `D` | Вправо |
| `E` | Вгору |
| `Q` | Вниз |
| `←` `→` | Рух ліво / право (як A / D) |
| `↑` `↓` | Рух вперед / назад (як W / S) |
| **Shift** (утримувати) | Повільний рух (0.3×) |
| **Ctrl** (утримувати) | Швидкий рух (sprint) |

---

## Обертання камери (клавіатура)

| Клавіша | Дія |
|---------|-----|
| `Numpad 4` | Повернути камеру вліво по горизонталі (yaw −15°) |
| `Numpad 6` | Повернути камеру вправо по горизонталі (yaw +15°) |
| `Numpad 8` | Повернути камеру вгору по вертикалі (pitch −15°) |
| `Numpad 2` | Повернути камеру вниз по вертикалі (pitch +15°) |

> **Numpad Emulation:** якщо увімкнено `setNumpadEmulation(true)`, цифрові клавіші верхнього ряду (`0`–`9`) та `-` / `=` / `.` замінюють Numpad.

---

## Масштаб (zoom)

| Клавіша | Дія |
|---------|-----|
| `Numpad +` | Наближення |
| `Numpad −` | Віддалення |
| Колесо миші ↑ | Наближення |
| Колесо миші ↓ | Віддалення |

---

## Перемикання режиму камери

| Клавіша | Дія |
|---------|-----|
| `Numpad 5` | Перемкнути FreeLook ↔ Orbit |

---

## Скидання і фокус

| Клавіша | Дія |
|---------|-----|
| `Numpad 0` | Камера-вид (зберегти поточний вид) |
| `Shift + C` | Скинути вид повністю (позиція + обертання) |
| `Alt + R` | Скинути обертання камери |
| `Alt + G` | Скинути позицію камери |
| `Numpad .` | Сфокусуватись на виділеному об'єкті |
| `Home` | Сфокусуватись на всіх об'єктах сцени |

---

## Виділення об'єктів

| Клавіша | Дія |
|---------|-----|
| **LMB** (клік) | Вибрати об'єкт під курсором |
| **Shift + LMB** | Додати / прибрати об'єкт з виділення (toggle) |
| `A` | Вибрати всі об'єкти |
| `Alt + A` | Зняти виділення з усіх |

---

## Трансформація об'єктів

Натиснути `G` → перемістити мишу → підтвердити або скасувати.

| Клавіша | Дія |
|---------|-----|
| `G` | Почати переміщення (Grab) |

**Обмеження по осі** (після `G`):

| Клавіша | Дія |
|---------|-----|
| `X` | Обмежити по осі X |
| `Y` | Обмежити по осі Y |
| `Z` | Обмежити по осі Z |

**Підтвердження / скасування:**

| Клавіша | Дія |
|---------|-----|
| `Enter` або **LMB** | Підтвердити трансформацію |
| `Esc` або **RMB** | Скасувати (повернути вихідний стан) |

---

## Дії з об'єктами

| Клавіша | Дія |
|---------|-----|
| `Shift + D` | Дублювати виділений об'єкт (відразу входить у Grab) |
| `X` або `Delete` | Видалити виділений об'єкт |

---

## Миша — Orbit / Zoom

| Дія миші | Результат |
|----------|----------|
| **RMB** + drag | Обертання камери (yaw/pitch) — FreeLook |
| **MMB** + drag | Обертання навколо цілі (orbit) |

### Emulate 3-Button Mouse (`setEmulate3ButtonMouse(true)`)

Для тачпадів та мишей без середньої кнопки:

| Дія миші | Результат |
|----------|----------|
| `Alt + LMB` + drag | Orbit навколо виділеного об'єкта |
| `Alt + RMB` + drag | Zoom (drag вгору = наближення) |

---

## Інші

| Клавіша | Дія |
|---------|-----|
| `B` | Увімкнути / вимкнути billboard-режим для об'єктів |
| `R` (без трансформ-режиму, поза текст-редактором) | Увімкнути / вимкнути візуалізацію ray |
| `Ctrl + C` / `Ctrl + X` / `Ctrl + V` | Копіювати / вирізати / вставити (текстовий редактор) |
| `Home` / `End` | Початок / кінець рядка (текстовий редактор) |
| `Enter` | Новий рядок (текстовий редактор) |
| `Escape` | Закрити текстовий редактор / скасувати трансформацію |
